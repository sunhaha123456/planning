流程模板：
    1、新建后，不能修改流程模板名
    2、冻结后：不能启动流程
    3、冻结后，才可以新增、编辑、删除节点
    4、有关联流程时，不能删除流程模板
       无关联流程时，可以删除流程模板
    5、只有一个流程节点的流程模板，也可以启动流程
    6、不允许有多个一级流程节点
=====================================================================================
流程实例：
    1、流程实例的名称、节点，都不能修改
    2、流程被管理员冻结后，流程节点将不会被用户看到
    3、已完成的流程，用户不能撤回、删除
                     管理员不能冻结、删除
    4、会签模式下，用户a提交了，用户b进行退回时，会将用户a的操作进行撤回
=====================================================================================
流程实例状态流程图：
    0（已启动）—>1（已完成）
               —>2（用户撤回）—> 删除
               —>3（管理员冻结）—> 删除
